🧭 Final Plan – CourseTrack (Multi-User, Cloud Hosted, Professional UI)
🌍 Core Concept

একটা full-stack web app যেখানে যে কেউ account খুলে নিজের course journey ট্র্যাক করবে —
Add/Edit/Delete course, note রাখবে, certificate লিংক দেবে, status mark করবে।
সব কিছু personal dashboard-এ stored থাকবে (Supabase DB-তে)।

⚙️ Tech Stack (Production Level)
Layer	Tech	Purpose
Frontend	React (JS) + TailwindCSS + shadcn/ui	UI + routing + responsive design
Backend	Supabase (PostgreSQL + Auth + Storage)	Serverless backend (no need Express manually)
Auth	Supabase Auth (Email + Google OAuth)	Multi-user secure login/signup
Database	Supabase built-in Postgres	All users’ & courses’ data store
Hosting	Vercel (frontend) + Supabase (backend/db)	Scalable, easy deploy
Optional API layer	Node/Express (later)	যদি custom server logic দরকার হয়

🔑 Database Schema (Supabase SQL)
-- users handled by supabase auth automatically

create table courses (
  id uuid primary key default uuid_generate_v4(),
  user_id uuid references auth.users(id) on delete cascade,
  title text not null,
  platform text,
  course_url text,
  certificate_url text,
  status text check (status in ('Planned','In-Progress','Completed')),
  start_date date,
  end_date date,
  notes text,
  created_at timestamp default now()
);
